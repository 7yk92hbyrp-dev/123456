THE ARSENAL

1. Unzip this at the root of your repo.
   The commands land in .claude/commands/, the tested code in code/,
   backend.sql sits beside them. (.claude is a hidden folder: press
   Cmd+Shift+. in Finder to see it. It is there.)
2. Open Claude Code in that repo.
3. Run /logger

That is the whole install. The exact tested logger, every brick,
wired to one saved table. Re-download after each episode: the same
/logger, grown.
